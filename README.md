# Accessily – Accessible Design Landing Page


## Project Overview


During this SBA I have been able to put accessibility into my actions more and more. In this project I was able to practice web design and foundational accessibility principles.

---


## Accessibility Considerations


Accessibility was a central focus throughout the development of this project. While building the site, I considered how users with different abilities might interact with the interface. Things like this are important to me; I believe everyone should be able to have an easy and pleasant user experience. And accessibility makes it possible. I had to consider many aspects of accessibility, such as ARIA labels for my forms and alt text for my images. These principles make all users feel welcome.
